# Evidence Bundles / Pacotes de Evidência

## Português

Cada diretório, exceto `TEMPLATE`, corresponde a uma contribuição auditável. O
manifest prende a mudança ao commit-base, ao `environment.lock.json`, aos
scripts de reprodução, aos relatórios e aos arquivos alterados por SHA-256.
Depois do merge, o pacote é histórico e não deve ser reescrito.

Copie `TEMPLATE`, execute a esteira local e siga [CONTRIBUTING.md](../CONTRIBUTING.md).

## English

Every directory except `TEMPLATE` represents an auditable contribution. Its
manifest binds the change to the base commit, `environment.lock.json`, replay
scripts, reports, and changed files using SHA-256. After merge, the bundle is
historical and must not be rewritten.

Copy `TEMPLATE`, run the local pipeline, and follow [CONTRIBUTING.md](../CONTRIBUTING.md).
